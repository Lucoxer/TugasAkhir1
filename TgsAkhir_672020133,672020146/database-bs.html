DATABASE

Definisi Singkat

A. Data Base

Data Base terdiri dari 2 kata, yaitu Data dan Base.

  • Data adalah representasi dari fakta dunia nyata yang mewakili suatu objek manusia.
  • Base adalah Gudang atau markas untuk berkumpul.
Dari kalimat diatas bisa disimpulkan, Data Base adalah Tempat penyimpanan atau gudang yang dapat menyimpan data dari fakta dunia nyata. Data base bisa digambarkan seperti brankas yang berfungsi sebagai menyimpan segala informasi atau hal penting dan dimasukkan kedalam brankas yang dimana membutuhkan kunci atau password untuk membuka dan menguncinya agar terjaga. Data Base dan Brankas sebenarnya memiliki tujuan yang sama atau prisnip yang sama yakni;
  • Memudahkan seseorang untuk mengambil atau menyimpan data dari brankas atau penyimpananan
  • Perbedaanya hanyalah pada media penyimpanan yang digunakan untuk menyimpan data. Perbedaan ini akan memunculkan segala metode/cara yang dapat digunakan dalam upaya penyimpanan.

B. Pengertian Data Base

  • Data Base adalah Kumpulan data yang saling berhubungan dan disimpan secara bersama sedimikian rupa dan tanpa pengulangan (redundansi) yang tidak perlu, untuk memenuhi kebutuhan.
  • Data Base adalah Kumpulan file yang mempunyai kaitan antara satu file dengan file yang lain sehingga membentuk satu bangunan data untuk menginfromasikan satu perusahaan, instansi dalam batasan tertentu.

C. Operasi Dasar Basis Data

Perbedaan Disk dan Data base ialah, jika disk kita dapat menempatkan beberapa(lebih dari satu) data base. Sedangkan Data Base, Kita dapat menempatkan satau atau lebih file atau tabel. Pada file/tabel ini sesungguhnya data dapat disimpat dan ditempatkan, misalnya Data Base kepegawaian, akademik dan pergudangan. Karena itu, operasi - operasi dasar yang dapat kita lakukan berkenaan dengan basis data dapat meliputi:

  • Pembuatan Basis data baru (create database),
  • Penghapusan basis data (drop database),
  • Penambahan/pengisisan data baru ke sebuah file/tabel di sebuah basis data (insert),
  • Penghapusan,pengambilan,dan pengubahan data dari sebuah file/tabel (delete,retrieve/search,and update),
  • Penghapusan dan pembuatan file/table baru ke suatu data base (drop table and create table)
