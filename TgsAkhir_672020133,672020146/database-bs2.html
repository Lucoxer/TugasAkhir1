D. Tujuan Data Base

Tujuan utamanya menemukan data yang kita cari dengan mudah dan cepat. Pemanfaatan basis data dilakukan untuk memenuhi sejumlah tujuan(objektif) seperti berikut ini :

  • Speed : diharapkan pengguna dapat melkaukan pentimpanan, perubahan dan menampilkan kembali dengan cepat dan mudah.
  • Space : Penggunanaan data base dapat mengurangi redundansi data.
  • Accuracy : Dapat menjaga keakuratan data lebih terjaga dengan menerapkan aturan data dan batasan tertentu.
  • Ketersediaan : penghapusan atau memindahkan ke media backup untuk menghemat ruang penyimpanan.
  • Kebersamaan : Data yang dikelola oleh sistem mendukung lingkungan multiuser(banyak pemakai)dengan menjaga/menghindari munculnya problem baru.

E. Penerapan Data Base

Data Base sudah banyak diterpakan di berbagai bidang seperti di bidang fungsional dan Organisasi / Perusahaan. Contoh bidang fungsional yang sudah memanfaatkan data base :

  • Kepegawaian
  • Akuntansi
  • Reservasi
  • Layanan Pelanggan (customer care)

Sedangkan dalam biang Organisasi/Perusahaan sebagai berikut :

  • Perbankan : melakukan pengelolaan data nasabah/data tabungan/data pinjaman dan pembuatan laporan
  • Produsen Barang : melakukan pengelolaan data keluar masuk barang
  • Industri Manufaktur : membantu dalam pengelolaan pesanan barang dan mengelola data karyawan.
  • Telekomunikasi
  • Pendidikan

E. Sistem DataBase

Sistem adalah Tatanan yang terdiri atas beberapa jumlah komponen fungsional yang saling berhubungan dan basis data hanyalah sebuah objek yang pasif/mati. Karena basis data ada pembuatnya dan tidak akan pernah berguna jika tidak ada penngelola, penggeraknya ialah sebuah program/ software. Jika keduanya digabungkan basis data dan pengelola akan menjadi sebuah sistem. Sistem Basis Data adalah Sistem yang terdiri dari kumpulan file yang saling berhubung dan memungkinkan untuk mengakses dan memanipulasi file file. Komponen-komponen utama pada Sistem Basis Data :

  • Perangkat Keras (Hardware),
  • Sistem Operasi(Operating System)
  • Data Base,
  • Sistem (aplikasi) Pengelola Data Base (DBMS),
  • User(Pemakai),
  • Aplikasi (perangkat lain) tetapi bersifat opsional.
